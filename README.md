# react-native-sunmi

A React Native library for Sunmi POS devices, enabling printing, barcode scanning, and native hardware integration

## Installation


```sh
npm install react-native-sunmi
```


## Usage


```js
import { multiply } from 'react-native-sunmi';

// ...

const result = multiply(3, 7);
```
